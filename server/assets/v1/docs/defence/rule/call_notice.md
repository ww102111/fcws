关于扎实做好廉洁征兵工作的通知
===

各军区、各军兵种、各总部、军事科学院、国防大学、国防科学技术大学、武警部队：

为认真贯彻党中央、中央军委和习主席关于征兵工作的重要决策指示，确保今年新兵征集工作廉洁顺利进行，进一步纯正征兵风气、提高兵员质量，现就做好廉洁征兵工作提出如下要求：

　　一、深化思想认识，切实把廉洁征兵工作作为一项严肃的政治任务抓紧抓好。兵员征集是建设强大军队的基础工程。廉洁兵
征高素质兵，事关党、国家和军队建设发展的根基，事关从兵之初写好军旅第一笔、从源头纯洁巩固部队，事关把部队建成锻造人培养人的大熔炉大学校，事关军队在社会和人民群众心中的形象、赢得人民群众的支持和信赖，对于提高部队凝聚力战斗力、实现党在新形势下的强军目标具有十分重要的意义。近年来，各级大力加强征兵廉政建设，强化监督检查，从严查处违纪违法案件问题，促进了征兵风气的好转。但也要看到，征兵工作中的腐败现象和不正之风依然存在，有的把征兵当作个人营私谋利的机会，私自截留、挪用、调整征集名额，搞权钱交易；有的在报名、体检、政审、定兵等环节吃拿卡要，甚至索贿受贿；有的领导机关打招呼、递条子，搞关系兵、人情兵，严重影响了兵员质量和军队形……象。这些问题也反映了一些单位和人员对廉洁征兵重要地位作用认识不够到位，廉洁征兵制度规定执行不够严格，监督防控还存在漏洞和薄弱环节，必须认真解决、坚决纠治。各级要从政治和全局的高度，充分认清做好廉洁征兵工作的极端重要性，自觉把廉洁征兵、征高素质兵作为征兵工作的重中之重，作为党的群众路线教育实践活动成效的实际检验，作为加强作风建设的重要抓手，以强烈的政治意识、责任意识和忧患意识，强化使命担当，积极顺应大势，大力倡导新风正气，切实抓紧抓实抓出成效。

　　二、严格纪律要求，进一步规范征兵工作秩序。坚决落实国家关于新兵征集的政策规定，认真贯彻2010年总参、总政和国
监察部印发的《关于加强廉洁征兵工作的意见》，严格执行公开公示、回避、责任追究等廉洁征兵工作制度，真正把各项纪律规定落到底、抓到位,把从严要求体现和贯穿到兵员征集报名、体检、政审、定兵等各个环节。征兵过程中，必须坚决贯彻执行“五条禁令”：严禁以权谋私，擅自截留、挪用和私分征集名额；严禁弄虚作假，在报名、体检、政审中填报虚假信息、提供不实证明；严禁吃拿卡要，以任何理由收取应征青年及其家属钱物；严禁暗箱操作，不按规定程序、权限私自定兵；严禁滥用职权，违规插手干预征兵事务、随意调整兵员去向。各级要自觉把“五条禁令”作为不可逾越的“红线”，强化法纪观念，严明纪律约束，坚持有纪必执、有违必查，确保征兵工作规范有序开展。

　　三、广泛宣传教育，大力营造廉洁征兵的浓厚氛围。要深入搞好廉洁征兵宣传，充分运用广播、电视、报刊和网络
信公向全社会广泛宣传征兵工作纪律规定特别是“五条禁令”，形成遵规守纪、正风肃纪的强大声势。要把廉政教育贯穿征兵工作全过程，结合正在开展的第二批党的群众路线教育实践活动，区分兵役机关人员、体检医护人员、政审工作人员、接兵干部等，有针对性地开展专题教育，强化组织意识和纪律观念。要坚持抓早抓小、防微杜渐，对苗头性倾向性问题早发现、早处置，防止小问题变成大问题，切实筑牢思想道德和法纪防线。要注重总结宣扬廉洁征兵先进典型事迹，充分利用近年来查处的征兵违纪违法典型案例搞好警示教育，积极营造崇尚廉洁、遵规守纪的新风正气。

　　四、强化执纪监督，坚决以零容忍态度查处征兵中的腐败问题。各级党委要把惩腐肃纪、廉洁征兵作为党委工程、主官工程来
，负起主体责任，纪委履行好监督责任，各相关职能部门要履职尽责，充分发挥专门监督、群众监督、舆论监督等作用，实施全方位全过程监督，切实形成监督合力。各级要积极协调地方有关部门，成立廉洁征兵军地联合检查组，采取普遍检查、随机抽查、突击检查等方法，督导检查各级贯彻落实廉洁征兵制度规定情况，及时发现和纠治违规违纪问题。各单位明查暗访组也要加强对征接兵工作人员依法履职、秉公用权情况的监督检查，有效发挥在廉洁征兵中的作用。7月15日前，各军区、省军区要建立廉洁征兵举报电话、信箱，并报总参动员部、总政纪检部，统一在“全国征兵网”上公布；从8月1日起，军区、省军区纪检部门要安排专人24小时值守举报电话，受理群众举报。对发现的违纪违法问题线索，必须严查快办，一经查实严肃处理，指名道姓进行通报；对群众举报不受理、查处不认真造成严重后果的，对违反廉洁征兵制度规定导致责任退兵的，严肃追究当事人和相关领导责任，决不姑息迁就，使纪律真正成为带电的高压线。
